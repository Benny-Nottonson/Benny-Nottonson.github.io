[valtertennis.com](https://valtertennis.com) is a Qwik site built for a tennis company

It is built using Qwik, with CSS managed through Tailwindcss, and some UI components being brought in from DaisyUI

Authentication and data storage is handled through supabase, as well as AuthJS, all of which runs on top of node

This project was a contracted freelance site, and is still a work in progress, with the main features needed to be fully implemented
